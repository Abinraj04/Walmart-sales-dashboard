#  Walmart Sales Analysis Dashboard

##  Project Overview

This project presents an interactive **Power BI dashboard** analyzing Walmart sales data to uncover key business insights such as revenue trends, product performance, and customer behavior.

---

##  Objectives

* Analyze sales performance
* Identify top products and categories
* Understand customer patterns
* Discover trends across time

---

##  Tools Used

* Power BI
* Excel

---

##  Key Insights

* Total sales: **1.59M**
* Profit: **476K**
* Profit Margin: **30%**
* Los Angeles is top-performing city
